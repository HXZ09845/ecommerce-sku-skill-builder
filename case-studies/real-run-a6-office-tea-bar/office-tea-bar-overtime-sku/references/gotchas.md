# Gotchas

## G1: Opening Unit Leaks Later Selling Points

Symptom: U1 starts showing auto-fill, remote control, or fast heating.

Repair: Add `This clip only establishes the pain and product introduction` and end before the water action starts.

## G2: Environment Anchor Overwrites Action Video

Symptom: B-class action no longer follows the source video composition.

Repair: Make the environment reference background-tone only and state that `@Video1` strictly controls action, framing, timing, and viewpoint.

## G3: Continuous Hand And Liquid Motion Deforms

Symptom: pouring hand, vessel, or liquid becomes unstable.

Repair: Use intent cue plus result state. Show preparation, then cut to a stable completed result.

## G4: Detail Image Replaces Whole Product

Symptom: material close-up changes the entire appliance.

Repair: Mark detail references as texture/detail only and return to the whole-product identity anchor.

## G5: Fake Text Appears

Symptom: model generates wattage, brand labels, link text, or screen UI.

Repair: Move exact text to VO, subtitles, overlays, or post-production. Add `Do not generate brand text, numeric wattage, link text, or CTA stickers.`
