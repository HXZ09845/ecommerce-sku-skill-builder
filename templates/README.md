# Public Templates

Use these templates when starting a new ecommerce SKU Skill or prompt-plan.

Recommended order:

1. Fill `product-brief-template.md`.
2. Fill `selling-point-map-template.md`.
3. Fill `asset-manifest-template.json`.
4. Draft `prompt-plan-template.md`.
5. After generation, fill `take-review-template.json` and write lessons back into the SKU Skill.

These templates intentionally use public placeholder IDs such as `@Image1` and `@Video1`. Replace the descriptions with your own product facts and approved material notes, but do not add private local paths or raw client file names.
